# @tgulls/hello

A simple CLI that prints "Hello, npm!"

## Usage

Run directly without installing:

```sh
bunx @tgulls/hello
```

Or install globally:

```sh
bun install -g @tgulls/hello
hello
```

## Local Development

```sh
bun run build
bun link
bunx @tgulls/hello
```

## Publishing

```sh
npm login
npm publish --access public
```
